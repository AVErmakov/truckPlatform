В LoadTypeVehicleSeeder для каждой машины записали типы грузов
-------
В поисковом запросе флаг на отслеживание.
    Если он стоит то надо запрос записать в search_settings.
    Для формирования поискового запроса исп-ть newSearchRequest($num).
Контроллер запускает метод checkSearch() в котором перебираем все грузы, проверяем
    и выводим коллекцию с добавленным ?атрибутом? profit
Для проверки каждого груза метод LoadService/singleLoadSearchCheck(SearchSetting, Load)
1.  Внутри метода проверяется соответствующий тип груза методом checkLoadType().
    Для этого взяли из запроса vehicle_id, по нему нашли все типы грузов,
    допустимых для данной машины. Выбрали их из таблицы loads.
2.  Внутри метода также проверяется дата и время загрузки.
    Это метод checkDatePickUpcheckDatePickUp().
3.  Проверяем на эффективность в методе checkProfit().
При появлении нового груза запускается (observer или еще как-нибудь) метод
    LoadService/singleLoadSearchCheck(SearchSetting, Load) и выводится такая же
    коллекция как и в случае с поиском.
    Если profit < 0 выводится пустая коллекция
